    Editor Help

    What's this editor?

    This is a Markdown editor, which allows you to write the newsletter using Markdown. There are also a variety of shortcut buttons for common actions, such as creating headers, images and formatting text. If you're unfamiliar with Markdown but familiar with Discord formatting, you actually know some Markdown already - formatting text is done exactly the same way, e.g. **bold**, *italics* and so on! Otherwise I'd recommend looking at a Markdown tutorial such as this one.

    What are classes?

    One of the rarer operations supported by our Markdown renderer is custom classes. These are added to various elements to change how they are rendered. You can add one of these using the {.class-name-here} syntax, or by clicking the custom classes button.

    In general, you shouldn't have to add your own custom classes as the most common ones have buttons provided. However, there are a few that you might want to add, which are listed below.

    • .news-bg adds a background to the associated header. You should use this with every heading, and if you click the Header button it will do this for you. An example is as follows: ## My Great Heading {.news-bg-1} - note the space between the heading and class.
    • .news-image-left and .news-image-right make an image float to that side and take up half of the width. This allows text to appear next to an image! There are also buttons on the toolbar that do this. An example is as follows: ![my great image](url){.news-image-left} - note the lack of a space between the iamge and class. (I don't know why this differs, but we'll just have to deal with it.)
    • .news-width-1 to .news-width-5 set the size of an image. (1 is the smallest.) You can combine this with other classes: {.news-image-left .news-width-1} both makes the image float to the left and sets its width.
    • .clearfix creates a section break. (There's also a button for this.) This means that if the text next to an image is shorter than the image, blank space will be left below that text to make the next text start after the bottom of the image.

    The Web Admin has details of this in their guide, so please ask them if you have any issues!
